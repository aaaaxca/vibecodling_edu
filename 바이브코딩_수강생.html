💻
나만의 웹사이트가 필요한 학생
개발·디자인 진로를 고려하며 자신의 프로젝트와 학업 스펙을 담을 반응형 개인 포트폴리오 사이트를 구글 AI 기술로 완성하고 GitHub Pages로 배포합니다. 실전 프론트엔드 UI 개발 프로세스를 경험합니다.
🙋
아직 나를 보여줄 공간이 없는 모든 사람
전공자가 아니어도 괜찮습니다. 1인 기업·프리랜서·부업의 시대, 나를 빠르게 어필할 링크 하나가 곧 경쟁력입니다. AI와 함께라면 코딩 지식 없이도 나만의 소개 페이지를 지금 바로 만들 수 있습니다.
🖼️
개인 작업물이 풍부한 학생
수행평가 자료, 디지털 일러스트, 소장용 사진, 직접 만든 코드까지 한눈에 모아 보여주는 맞춤형 갤러리를 제작합니다. 자신만의 디지털 보관소를 구축합니다.
🎨
예술·테크 융합 관심 학생
시각 디자인 감각과 아이디어를 구글 AI 생태계 및 최신 미디어 기술에 융합하는 크리에이티브 디렉팅을 경험합니다. 기술적 한계를 뛰어넘는 창작이 가능합니다.

🎯 과정 취지

AI를 코딩 비서와 디자인 파트너로 활용하여 **자신의 커리어와 프로젝트를 담은 웹 포트폴리오를 직접 기획·제작·배포하는 경험**. 복잡한 백엔드 인프라 장벽 없이 Google AI Studio와 Gemini로 컴포넌트를 조율하고, NanoBanana로 독창적인 비주얼 에셋을 생성하여 GitHub Pages로 라이브 배포합니다.

📅 주차별 커리큘럼
주차주요 내용학습 목표 (이론 + 실습 융합)
1주차 포트폴리오 기획 & 비주얼 에셋 생성 Gemini 연동 사이트 컨셉 기획 및 레이아웃 구조 설계 · NanoBanana를 활용한 포트폴리오 무드보드 빌딩 및 고품질 배경/프로필/아이콘 시각 에셋 제작
2주차 AI Studio UI 와이어프레임 설계 Google AI Studio의 System Instruction(시스템 지침) 설정을 통한 최적의 코딩 비서 프롬프트 튜닝 · 레이아웃 가이드라인 도출 및 핵심 컴포넌트(Header, Profile) 와이어프레임 빌딩
3주차 멀티모달 컴포넌트 & 인터랙션 구현 AI Studio 멀티모달(시각+텍스트) 기능 활용: 레퍼런스 디자인 이미지를 기반으로 클라이언트 사이드 고도화 · 프로젝트 필터링 탭, 다크모드, 부드러운 스크롤 애니메이션 등 인터랙티브 요소 강화

🎨
수동적 학습에서 '총괄 디렉터' 경험으로
단순히 강사의 코드를 따라 치는 방식이 아닙니다. 학생이 아이디어를 던지면 AI가 조력자가 되어 자신의 기획을 고품질 포트폴리오와 미디어로 실현해 내는 리더십 교육입니다.
🤖
AI 멀티모달 기술의 선제적 마스터
단순한 챗봇 검색 수준을 뛰어넘어, 구글의 차세대 프롬프트 개발 도구를 유기적으로 연동하고 조율하는 '멀티모달 프롬프트 오케스트레이션' 능력을 선점하여 미래 경쟁력을 확보합니다.
📂
가장 트렌디한 포트폴리오 내재화
누구나 제출하는 흔한 과제물이 아닌, 실제 라이브 도메인으로 배포된 개인 웹사이트와 화려한 디지털 아카이브 작품이 추출되므로 대입 자소서 및 수행평가에서 압도적인 가점을 받습니다.

FAQ

자주 묻는 질문

수강 전 궁금하신 것들을 미리 정리했습니다.

Q. 바이브코딩 1과 2는 어떻게 다른가요?
1단계가 전체적인 레이아웃과 소개를 잡는 '웹 포트폴리오 입문'이라면, 2단계는 수많은 데이터와 이미지를 감각적으로 배치하고 컨트롤하는 '아카이브 갤러리 심화' 과정입니다. 동일한 구글 AI 도구를 더 고차원적으로 활용하며 실무 수준의 복합 그리드와 인터랙션을 완성합니다.
Q. 갤러리에는 어떤 작업물을 올릴 수 있나요?
수행평가 보고서, 직접 그린 디지털 일러스트, 소장 중인 사진, 직접 작성한 코드까지 종류에 상관없이 모든 창작물을 담을 수 있습니다. 여기저기 흩어져 있던 소중한 작업물들이 세상에 하나뿐인 온라인 갤러리로 완성됩니다.
Q. 왜 GitHub Pages로 배포하나요?
GitHub은 전 세계 개발자들이 코드를 저장하고 협업하는 표준 플랫폼입니다. AI로 만든 결과물을 실제 본인의 GitHub 저장소에 업로드하고 무료로 웹 배포까지 연결하기 때문에, 입시나 면접에서 포트폴리오의 진정성을 직접 증명할 수 있습니다.
Q. 결과물이 입시나 진로에 도움이 되나요?
네, 실질적으로 유리합니다. GitHub Pages로 실제 배포되어 전 세계 어디서나 링크 하나로 접속할 수 있는 디지털 갤러리 포트폴리오는 자소서와 수행평가, 면접에서 강력한 차별점이 됩니다.
Q. AI가 다 만들어주면 내가 배우는 게 있을까요?
AI는 기획을 빠르게 실현해주는 조력자입니다. 어떤 구조로 사이트를 구성할지, 어떤 정보를 어떻게 배치할지는 온전히 본인의 몫입니다. 웹의 구성 원리를 함께 익히기 때문에, 수강 후에는 AI에게 더 정확하게 요청하고 결과물을 빠르게 완성하는 능력도 자연스럽게 갖추게 됩니다.
Q. 수강 순서는 어떻게 되나요?
바이브코딩 1(웹 포트폴리오, 4주 · 월~목)을 먼저 수강한 후, 바이브코딩 2(아카이브 갤러리, 4주 · 월~목)로 이어지는 연속 코스를 추천합니다. 1단계에서 다진 기반 위에 2단계가 자연스럽게 확장됩니다.
